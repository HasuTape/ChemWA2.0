Rodzaje reakcji chemicznych

Reakcje chemiczne mają różne rodzaje w zależności od tego, co się w nich dzieje.

  • Chemia nieorganiczna
    • Reakcje syntezy - kilka substancji tworzą jedną (A + B → AB)
    • Reakcje analizy - jedna substancja tworzy kilka (AB → A + B)
    • Reakcje wymiany - wymiana składników substancji (A + BC → AC + B)
    • Reakcje wymiany pojedynczej - wymiana jednej grupy/pierwiastka w związku chemicznym (A + BC → AC + B)
    • Reakcje wymiany podwójnej - wymiana dwóch grup/pierwiastków w związkach chemicznych (AB + CD → AD + CB)
  • Chemia organiczna
    • Addycja - pęknięcie jednego wiązania wielokrotnego i przyłączenie podstawników, np. CH2=CH2 + Cl2 → ClCH2-CH2Cl
    • Eliminacja - odłączenie się podstawników i powstanie wiązania wielokrotnego, np. ClCH2-CH2Cl → CH2=CH2 + Cl2
    • Substytucja - zamiana jednego podstawnika na innych
      • Substytucja nukleofilowa - podstawnik jest zastępowany przez jon lub cząsteczkę o wolnej parze elektronowej (nukleofil)
      • Substytucja elektrofilowa - podstawnik jest zastępowany przez jon lub cząsteczkę z deficytem elektronowym (elektrofil)
      • Substytucja rodnikowa - podstawnik jest zastępowany przez rodnik (cząsteczkę z niesparowanym elektronem), który powstaje np. wskutek działania promieniowania UV rozbijającego wiązania kowalencyjne.

Zaznacz typ reakcji: S - synteza, A - analiza, W - wymiana, E - eliminacja, Ad - addycja, Sb - substytucja.

  1. Al + CuSO4 → Al2(SO4)3 + Cu
  2. 2H2O2 → 2H2O + O2
  3. H2O + SO2 → H2SO3
  4. CH2=CH-CH3 + HBr → CH3-CHBr-CH3
  5. CH3-CHCl-CH2Cl → CH3-CH=CH2 + Cl2
  6. CH3-CH3 + Cl· → CH3-CH2Cl + H·
